\begin{frame}{\probno{}. \probtitle{}}
    \begin{itemize}
        \item 룬(정점)의 마나(가중치)를 업데이트하는 쿼리와, 연결된 회로(차수)가 $d$개인 모든 룬의 인접 마나 총합을 구하는 쿼리를 처리해야 합니다.
        \item 매 쿼리마다 그래프를 직접 탐색하면 $O(Q \times M)$으로 TLE를 받습니다.
        \item 임의의 그래프에서 모든 정점의 차수의 합은 $2M$입니다.
        \item 따라서 한 정점의 인접 정점들이 가질 수 있는 서로 다른 차수의 개수는 최대 $O(\sqrt{M})$개입니다.
        \item 각 정점마다 인접한 정점들을 map을 사용하여 pair\{차수 $d$, 개수 $cnt$\} 형태로 저장합니다.
        \item 각 1번 쿼리가 들어올 때마다 map에 존재하는 $O(\sqrt{M})$이하의 pair만 순회하며 답을 업데이트 합니다.
        \item 시간복잡도는 $O(M\log{N} + Q\sqrt{M})$으로 문제를 해결할 수 있습니다.
    \end{itemize}
\end{frame}
