\def\probtitle{최대 기울기 증가 부분수열}
\def\probno{L} % 문제 번호

\begin{problem}{\probtitle}{표준 입력(stdin)}{표준 출력(stdout)}{2.5 초}{1024 megabytes}

2차원 평면 위에 $N$개의 점이 주어진다. 이 중 일부를 선택하여 수열 $S$를 구성한다.

수열 $S$의 $i$번째 원소를 $S_i = (x_i, y_i)$라고 할 때, $S$는 다음 조건을 만족해야 한다.
\begin{itemize}[topsep=0pt,noitemsep]
\item 모든 $1\leq i < |S|$에 대해 $x_i < x_{i+1}$이고 $y_i < y_{i+1}$이다.
\item 모든 $1\leq i < j < k \leq \lvert S\rvert$에 대해 $\dfrac{y_j - y_i}{x_j - x_i} \leq \dfrac{y_k - y_j}{x_k - x_j}$이다.
\end{itemize}

\InputFile

첫째 줄에 점의 개수 $N$이 주어진다.

다음 $N$개의 줄에 각각 점의 $x$좌표와 $y$좌표를 의미하는 두 정수 $x$, $y$가 공백으로 구분되어 주어진다.

\OutputFile

$S$의 최대 길이를 출력한다.

\Constraints

\begin{itemize}[topsep=0pt,noitemsep]
    \item 주어지는 모든 수는 정수이다.
    \item $1 \le N \le 3\,000$
    \item $-10^9 \le x,y \le 10^9$
    \item 같은 좌표를 가지는 점은 없다.
\end{itemize}

\Example

\begin{example}
    \exmpfile{./example/01.in.txt}{./example/01.out.txt}%
    \exmpfile{./example/02.in.txt}{./example/02.out.txt}%
    \exmpfile{./example/03.in.txt}{./example/03.out.txt}%
\end{example}

\end{problem}
